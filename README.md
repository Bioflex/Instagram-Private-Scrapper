# Instagram-Private-Scrapper

This Python application is a GUI-based Instagram Scraper built with CustomTkinter and Instaloader. It allows users to securely log in to their Instagram account, handle session files, and scrape media from any public or accessible profile. The tool organizes downloaded posts into separate images and videos folders and supports automatic .webp â†’ .jpg conversion.
## Deployment

To deploy this project run go to the project directory and run-

```bash
  python scraper.py
```
